Projects

Stock Dashboard | Python, Dash, Plotly, HTML/CSS

This is a stock analysis program made with Python using Dash. Originally I was going to make a stock watchlist app but decided against it. This project gave me an opportunity to improve at HTML/CSS and web app design in general. I also learned more about data visualization in Python which is one of the biggest uses for the language today. I improved at pulling data from API's and using Pandas and similar libraries to manipulate it. There is a section for company metrics, a price chart, an analyst sentiment graph, a volume bubble chart, and a news module.

Odd Jobs | Unity, C#

This is a short horror game made in Unity. It was my first full 3D game and it changed forms a few different times. I learned some lessons about the importance of thorough pre-production planning and attempting to stick to the plan to avoid endless feature creep. I think some iteration while designing is a good thing but I wasted a lot time reworking things in hindsight. Main project components were level design, first person controller, interactions by raycast, horror design/events, and monster AI.

Game Jam / Class Projects

Lava Escape | UE4, Blueprints

Lava Escape is a puzzle platformer game made in Unreal for a gameplay programming course. You have to keep track of the room's layout and change elements between mushrooms that give you a double jump and clouds that disappear after a few seconds. This was my first Unreal game and helped me learn the editor and general workflow. I also learned more about level design and puzzle design in particular.

UATanks | Unity, C#

UATanks is a multiplayer tank arena game made in Unity. It was made in 5 weeks for a programming course and I learned a lot in that time. There is a procedural level generator to randomize the map's layout. The enemies have state machines for decision-making and each of the four AI has unique behavior. I added split-screen multiplayer but it was trickier than expected. I ran into some interesting issues like considering where to put the only audio listener when both players needed to be able to hear at the same volume.

Thief in the Night | Unity, C#

I made this game for the Itch.io Weekly Game Jam #235 for the Bandit theme. I learned a few things making this game but the biggest was to prototype the game mechanics to make sure the gameplay is solid / fun before making art and assets. I made that mistake and as a result the first iteration was too easy and boring. I ended up improving the thief AI and changing the game so that you only had four knives which had to be picked up on hit or miss. It's a better gameplay experience than when I started and it taught me that the core gameplay loop is the most important thing to consider.
