Game Jam / Class Projects

Lava Escape | UE4, Blueprints

Lava Escape is a puzzle platformer game made in Unreal for a gameplay programming course. You have to keep track of the room's layout and change elements between mushrooms that give you a double jump and clouds that disappear after a few seconds. This was my first Unreal game and helped me learn the editor and general workflow. I enjoyed visual scripting by Blueprint and look forward to doing more of that in the future.

UATanks | Unity, C#

UATanks is a multiplayer tank arena game made in Unity. It was made in 5 weeks for a programming course and I learned a lot in that time. A procedural level generator was created to randomize the map's layout. The enemy AI followed state machine logic for decision-making and one of them used waypoint pathfinding to navigate. I also added split-screen multiplayer which was trickier than expected. I ran into some interesting issues like considering where to put the sole audio listener when both players needed the same volume.
